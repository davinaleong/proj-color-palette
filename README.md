Tech Stack

- jQuery
- HTML/CSS

```JSON
                    {
                        "name": "Gray",
                        "hex": "#888888",
                        "rgb": "rgb(136, 136, 136)",
                        "hsl": "hsl(0, 0%, 53%)",
                        "text": "#000000"
                    },
                ]
            }
        ]
    }
]
```
